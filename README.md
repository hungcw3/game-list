# 🎮 Game List Manager

Ứng dụng web quản lý danh sách game cá nhân, cho phép theo dõi trạng thái các game bạn muốn chơi, đang chơi hoặc đã hoàn thành.

## Tính năng

- Thêm / xóa game
- Cập nhật trạng thái: Muốn chơi / Đang chơi / Đã xong
- Dữ liệu được lưu vào database

## Công nghệ sử dụng

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express
- **Database:** SQLite

## Cách chạy project

1. Clone repo về máy:
git clone https://github.com/hungcw3/game-list.git
2. Cài dependencies:
npm install
3. Chạy server:
node server.js
4. Mở trình duyệt tại `http://localhost:3000`
